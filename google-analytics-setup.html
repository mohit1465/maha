Google Analytics Setup Instructions

Steps to Complete Google Analytics Setup:

  1. Get Your GA Measurement ID:
    • Go to Google Analytics: https://analytics.google.com
    • Create a new property or use existing one
    • Find your Measurement ID (format: G-XXXXXXXXXX)
  2. Replace GA_MEASUREMENT_ID:
    • In all HTML files, replace "GA_MEASUREMENT_ID" with your actual ID
    • Example: gtag('config', 'G-1234567890');
  3. Add to All Pages:

    Add this tracking code to the section of every HTML page:

    <!-- Google Analytics (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=YOUR_ACTUAL_GA_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'YOUR_ACTUAL_GA_ID');
</script>
  4. Verify Setup:
    • Install Google Analytics Debugger Chrome extension
    • Visit your website and check browser console
    • Look for "Google Analytics" debug messages
  5. Check Real-time Reports:
    • In GA, go to Reports > Realtime
    • Visit your website
    • You should see active users within 30 seconds

  • Using gtag.js (latest version) instead of analytics.js
